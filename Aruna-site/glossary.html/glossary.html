AI Glossary

Important AI Terms Every Beginner Should Know

Artificial Intelligence (AI)

Technology that enables machines to perform tasks that normally require human intelligence.

Machine Learning (ML)

A branch of AI where computers learn from data without being explicitly programmed.

Deep Learning

A subset of ML that uses neural networks to process large amounts of data.

Neural Network

A computer system inspired by the human brain.

Dataset

A collection of data used to train AI models.

Algorithm

A set of rules or instructions used to solve problems.

NLP

Natural Language Processing helps computers understand human language.

Computer Vision

Allows computers to understand images and videos.

Generative AI

AI that creates text, images, music, videos, and code.

Chatbot

An AI system that interacts with users through conversation.
